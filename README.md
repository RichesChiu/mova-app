# Mova App

当前仓库使用 Xcode 模板推荐的工程目录结构：`Mova.xcodeproj` 位于仓库根目录，`Mova/` 是应用源码目录。

## 目录结构

```text
mova-app/
  Mova.xcodeproj
  Mova/
    AppRootView.swift
    MovaApp.swift
    AuthModels.swift
    AuthService.swift
    Media*.swift
    Assets.xcassets/
  API.md
  README.md
```

## 说明

- `Mova.xcodeproj`：Xcode 工程文件
- `Mova/`：应用源码目录
- `API.md`：当前服务端 API 对接说明

本地开发时，直接打开 `Mova.xcodeproj` 即可。
