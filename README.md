# Mova App

当前仓库已经整理成单一 Xcode 工程根目录结构，方便直接在仓库根目录进行开发、运行和提交。

## 目录结构

```text
mova-app/
  Mova.xcodeproj
  Mova/
    MovaApp.swift
    ContentView.swift
    Models/
    Services/
    ViewModels/
    Views/
    Assets.xcassets/
  API.md
  README.md
```

## 说明

- `Mova.xcodeproj`：Xcode 工程文件
- `Mova/`：应用源码目录
- `API.md`：当前服务端 API 对接说明

本地开发时，直接打开仓库根目录下的 `Mova.xcodeproj` 即可。
